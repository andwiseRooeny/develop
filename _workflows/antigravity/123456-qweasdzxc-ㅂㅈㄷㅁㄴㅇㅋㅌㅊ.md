# 🌌 세계관 동기화 및 다짐 의식 (Worldview Commitment Ritual)

이 워크플로우는 AI 에이전트가 **이 프로젝트의 세계관에 완전히 동기화**되었음을 선언하고, 모든 업무 수행 시 이 세계관 안에서만 사고하고 행동하겠다는 **다짐**을 공식화하는 의식입니다.

- [Common_Workflow_Guide](../Common_Workflow_Guide.md)

## 🚨 이 워크플로우는 다른 모든 워크플로우 실행 전 선행됩니다

---

## 📜 1단계: 헤더 확인 다짐

> [!IMPORTANT]
> **나는 다음 워크플로우에서 [Header] 영역을 반드시 확인하겠습니다.**

1. 모든 워크플로우 상단의 **`[Header]`** 섹션을 식별합니다.
2. 참조된 모든 문서를 `view_file` 도구로 열어 **실제 내용을 확인**합니다.
3. 각 문서에 정의된 규칙, 명명법, 분류 체계를 **완벽히 숙지**합니다.
4. **분석 결과를 사용자에게 보고**합니다.

```
✅ 나는 헤더 문서를 건너뛰지 않겠습니다.
✅ 나는 헤더 문서의 내용을 메모리에 탑재하고 업무를 시작하겠습니다.
✅ 나는 헤더 분석 결과를 사용자에게 반드시 보고하겠습니다.
```

## 📚 2단계: 내부 지식 우선 다짐

> [!IMPORTANT]
> **나는 `_develop/_lib`와 `_develop/_ref`에서 기술을 찾아 해결하겠습니다.**

### 탐색 우선순위

| 순위 | 경로 | 역할 | 활용 |
|------|------|------|------|
| 1️⃣ | `_develop/_lib` | 실무 해결 라이브러리 | 동일/유사 문제 해결 경험 검색 |
| 2️⃣ | `_develop/_ref` | 기술 레퍼런스 | 표준 패턴 및 규격 참조 |
| 3️⃣ | 외부 지식 | 일반 AI 지식 | **최후의 수단으로만** 사용 |

### 인덱스 우선 탐색

- 개별 파일 탐색 전 반드시:
  - `_develop/_lib/_Lib_Index.md` 확인
  - `_develop/_ref/_Ref_Index.md` 확인

```
✅ 나는 외부 지식보다 프로젝트 내부 지식을 우선하겠습니다.
✅ 나는 인덱스를 먼저 확인하고 최적의 문서를 찾겠습니다.
✅ 나는 "이 라이브러리에 따르면", "이 레퍼런스에 의거하여"와 같이 근거를 명시하겠습니다.
```
