## 1. 준비 단계
1. `_develop` 폴더가 존재하는지 확인합니다. 없다면 생성합니다.
   // turbo
   ```powershell
   if (-not (Test-Path "_develop")) { New-Item -ItemType Directory -Force -Path "_develop" }
   ```

## 2. 저장소 가져오기
사용자로부터 입력받은 Git 주소를 사용하여 `_develop` 폴더 내에 클론합니다.

1. **[INPUT]** Git 저장소 주소를 입력받습니다. (변수: `$GitUrl`)
2. 저장소 이름을 추출합니다.
3. `git clone` 명령어를 실행합니다.

```powershell
# 예시 스크립트 (사용자가 직접 실행하거나 승인 필요)
$GitUrl = "INPUT_GIT_URL_HERE" # 사용자가 입력한 URL로 대체됨
# 현재 시간을 이용해 고유한 임시 폴더명 생성
$TempDir = "_develop/_temp_clone_$(Get-Date -Format 'yyyyMMddHHmmss')"

Write-Host "Cloning from $GitUrl to $TempDir..."
git clone $GitUrl $TempDir

if (Test-Path $TempDir) {
    # .git 폴더 유지 (향후 git pull로 업데이트 가능)
    Write-Host "Moving contents to _develop..."
    # 임시 폴더의 모든 내용물(숨김 파일 포함)을 _develop으로 이동
    Get-ChildItem -Path $TempDir -Force | Move-Item -Destination "_develop" -Force

    # 임시 폴더 삭제
    Remove-Item $TempDir -Force
    Write-Host "완료: 저장소의 내용물을 _develop 폴더로 가져왔습니다."
} else {
    Write-Error "오류: Git Clone에 실패했습니다."
}
```
